EOPKG Ashtray

EOPKG ashtray is a custom private EOPKG repository, which holds all the packages not included by default in Solus repository.
This means it's actually built to be used only on Solus-project systems (or, at least, on the ones using eopkg / pisi package-manager.

Installation

The installation instructions are actually very straightforward: 

eopkg add-repo streambinder https://solus.davidepucci.it/eopkg-index.xml.xz
